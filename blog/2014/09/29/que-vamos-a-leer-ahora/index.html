Vamos a elegir entre todos que leer. Esta es la primer actividad de este tipo que tienen en clase. Vamos a elegir entre varias opciones y para eso es necesario conocerlas lo mejor posible.

Si bien vamos a leer 1 de los textos entre todos, si se quedan con ganas de leer algún otro seguro podrán hacerlo.

Entonces, estan son las instrucciones para elegir qué vamos a leer

  1. Leer de qué se tratan las tres opciones. Debajo de las instrucciones están los enlaces para cada opción.
  2. Una vez que leiste las opciones completar el formulario. El enlace está debajo de las opciones.
